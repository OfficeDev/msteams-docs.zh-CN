# <a name="designing-your-microsoft-teams-app-with-basic-fluent-ui-components"></a>使用基本的熟知 UI 组件设计 Microsoft 团队应用程序

团队应用程序通常是使用以下基本的熟知 UI 组件生成的。 这些组件尽可能地设计为平面化，可以跨不同的用例、主题和屏幕大小工作。

此页上的图显示了组件在团队中的外观默认 (光源) 和深色主题。

## <a name="microsoft-teams-ui-kit"></a>Microsoft 团队 UI 工具包

根据 <a href="https://fluentsite.z22.web.core.windows.net/" target="_blank">熟知的 UI</a>，MICROSOFT 团队 UI 工具包包含专门为构建团队应用程序而设计的组件和模式。 在 UI 工具包中，您可以将此处列出的组件直接插入到设计中，并查看有关如何使用每个组件的更多示例。

> [!div class="nextstepaction"]
> [ (Figma) 获取 Microsoft 团队 UI 工具包 ](https://www.figma.com/community/file/916836509871353159)
